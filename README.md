**功能介绍**<br>
<br>
1️⃣让标签搜索器中的搜索按钮来到**顶部**<br>
2️⃣提供快捷的**首字母**筛选器<br>
3️⃣支持**连续筛选**<br>
4️⃣支持**一键清空**选择的标签<br>
5️⃣适用于中文和英文<br>
<br>
⚠️仅在MAC和Zotero8上进行了运行测试<br>
<br>
**Feature Overview**<br>
<br>
1️⃣Move the search button to the **top** of the tag searcher<br>
2️⃣Provide a quick **initial letter** filter<br>
3️⃣Supports **sequential filtering**<br>
4️⃣Supports **clearing** selected tags with a single click<br>
5️⃣Works for both Chinese and English<br>
<br>
⚠️Just Test on MAC and Zotero8<br>
