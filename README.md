**功能介绍/Feature Overview**\n
让标签搜索器中的搜索按钮来到顶部，并提供快捷的首字母筛选器。点击字母可以快速筛出目标的标签。适用于中文和英文。\n
Move the search button to the top of the tag searcher and provide a quick initial letter filter. Clicking a letter allows you to quickly filter for the desired tag. Works for both Chinese and English.\n
